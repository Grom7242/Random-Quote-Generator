# Random-Quote-Generator

**Автор:** Азаров Захар

## Описание

Random Quote Generator — это простое GUI-приложение на Python для генерации случайных цитат. Программа позволяет:

- Генерировать случайные цитаты из предустановленного списка.
- Просматривать историю всех сгенерированных цитат с указанием времени.
- Фильтровать историю по автору или теме.
- Добавлять новые цитаты с проверкой на пустые поля.
- Сохранять историю и список цитат в JSON-файлы для сохранения данных между сеансами.

## Используемые технологии

- **Python 3.x**
- **Tkinter** — для графического интерфейса
- **JSON** — для хранения данных
- **Random** — для случайного выбора цитат
- **Git** — для контроля версий
